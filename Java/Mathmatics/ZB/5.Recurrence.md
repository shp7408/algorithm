### 점화식과 재귀함수
- 점화식과 재귀함수 개념 이해
- 점화식 유형을 파악하고 재귀함수로 구현할 수 있도록 한다.

<br>

# 점화식(Recurrence)
: 어떤 수열의 일반항을 그 이전의 항들을 이용하여 정의한 식  
예) 피보나치 수열
```
1, 1, 2, 3, 5, 8, 13, …
F1 = F2 = 1, Fn+2 = Fn+1 + Fn
```
앞의 두 수의 합이 그 다음 항이 된다.

<br>

# 재귀함수
: 어떤 함수가 자신을 다시 호출하여, 작업을 수행하는 방식
```
반환타입 함수이름 (매개 변수) {
	종료 조건
	…
	함수이름(…)
}	
```
